stateTransitions

Describes of how to transition from state to state. The current state corresponds to the image that will be shown (i.e. row from stimuli)

NameRequiredTypeDefault valueDescription
currentStateYesnumber1The state the program is currently in
positionYesnumber1The position needed to trigger the change (index of the positions in targetPosition)
distanceAllowedYesnumber1the maximum distance from the position that will trigger the change
penTouchingNonumber0whether the pen must be touching (If not using a tablet set to 0, for a tablet 0 = doesn't matter, 1 = must touch, 2 = must not touch)
timeElapsedNonumber0the minimum amount of time that has elasped since arriving in this state
newStateYesnumber1the state to move to if all the conditions are satisfied
