rectangle

A rectangle to be drawn in each frame

NameRequiredTypeDefault valueDescription
rectanglesFilenameNostring[]A file containing 5 elements per row - the x,y of the top-left corner, and the width and height of the rectangles to draw (in the stimuli directory), in the range 0 to 1. The fifth element is whether to fill the rectangle (0 or 1)
rectanglesNon by 5 matrix[]An array with the rectangle data (same format as for rectanglesFilename). Can specify either rectanglesFilename or rectangles
framesPerRectangleNonumber4number of frames to show each rectangle for
colorNo1 by 3 matrix[255,0,0]The color of the rectangle (RGB, 1x3 matrix, from 0 to 255)
