targetInSpace

The target(s) are locations in space

NameRequiredTypeDefault valueDescription
delayedFeedbackNonumber0Whether the feedback is to be shown in the following trial
checkGoingForwardNonumber0Whether it should be checked that the marker continually moves forward
targetsYesmatrix[]The possible targets for this trial
thresholdNonumber3How close to the target the marker needs to be to hit it
endtimeNonumber0The amount of time you need to be at the target for (in seconds)
dimensionsToUseNomatrixNaNDimensions to use to compare the target position , e.g. to just use x and y and not z, set to [1 2]. Default (NaN) is to use all dimensions. The number of dimensions should match the number of dimensions of the targets as defined in targetPosition. Alternatively, it can be set to how much to multiply the values with each row referring to one output dimension, e.g. if there are two xyz sensors, then [0.5 0 0 0.5 0 0;0 0.5 0 0 0.5 0]would generate an average of sensor 1 and 2. In this case, the number of rows should match the dimensions of the targets.
arrivalFeedbackStartNonumberNaNIf this number is not NaN, then when the target needs to arrive (in seconds) after to NOT get a "too early" feedback
arrivalFeedbackEndNonumberNaNIf this number is not NaN, then when the target needs to arrive (in seconds) before to NOT get a "too late" feedback
