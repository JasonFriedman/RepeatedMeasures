tablet

Connects to server to sample location and other information from a WACOM tablet

NameRequiredTypeDefault valueDescription
serverYstring'localhost'Location of the server (e.g. 'localhost' or an IP address)
portYnumber0Port to connect to the server on
maxxYnumber1Maximum x value in tablet coordinates (43600 for the Cintiq 21UX)
maxyYnumber1Maximum y value in tablet coordinates (32799 for the Cintiq 21UX)
showPositionTypeNstring'dot'If showing position, how to show the current position (either 'dot' or 'rectangle'). Note that if showPosition=1, it will always be shown. If showPosition=2 and imagestimulus is used,it will only be shown when in state 1
showPositionColorNn by 3 matrix[192,192,192]If showing position, the color of the feedback (each number between 0 and 255). If showPositionOnlyWhenTouching=0,the second row of this matrix can have the color for when the pen is not touching
showPositionSizeNmatrix6If showing position and showPositionType is a dot, the size of the dot (in pixels). If type is a rectangle, then a 1x2 array (width and height)
showPositionOnlyWhenTouchingNboolean''Whether to show position only when touching (0=also when not touching, 1=only when touching)
displayRangeXNn by 5 matrix[0,0,0,0,0]Range of values to show when using showPosition=3 as the dot position on the x axis. The ith row corresponds to the ith sensor. Use the first two values to show the mouse x values, next two for y. e.g. [-1 1 0 0 0] to show x values of -1 to 1 as the dot x position. Use the last value to show time, e.g. [0 0 0 0 1] will move across the screen in 1 s
displayRangeYNn by 5 matrix[0,0,0,0,0]Range of values to show when using showPosition=3 as the dot position on the y axis. The ith row corresponds to the ith sensor. Use the first two values to show the liberty x values, next two for y, e.g. [0 0 -10 10 0] to show y values of -0.5 to 0.5 as the dot y position. Use the last value to show time, e.g. [0 0 0 0 1] will move up the screen in 1 s.
offsetXNnumber0.50x offset when using showPosition
offsetYNnumber0.50y offset when using showPosition
