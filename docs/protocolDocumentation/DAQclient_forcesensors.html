forcesensors

This client connects to a DAQ server to sample force sensors.

NameRequiredTypeDefault valueDescription
numchannelsYesnumber[]Number of DAQ channels to sample
gainsYesmatrix1Gains (one per channel), usually in N / V
offsetsYesmatrix0Offsets (one per channel), usually in N
filterTypeNostring[]Filter to apply if showing position (current only option is 'butter' for Butterworth filter). Note that filtering is only for onscreen display, and the saved data is not filtered. Use of a filter requires the signal processing toolkit. An empty string means no filtering.
filterOrderNonumber4If using a filter, which order to use
filterCutoffNomatrix8If using a filter, the cutoff frequency for a lowpass filter (in Hz). If 2 numbers are specified, a bandpass filter will be created.
samplerateNonumber150Sample rate (in Hz), only needed if using a filter
