movementonset

Make targets appear / disappear based on some type of movement

NameRequiredTypeDefault valueDescription
typeYnumber1Type of movement onset to make targets appear / disappear (1 = movement threshold in a frame). Support must be provided by the stimulus or client.
directionNnumber-3direction to start moving when detecting movement onset (x=1,y=2,z=3), use negative values if appropriate.
thresholdNnumber0.10threshold to start moving when detecting movement onset (i.e. how far you need to move in a single frame)
