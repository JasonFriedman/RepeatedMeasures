liberty

Connect to the liberty to sample from it

NameRequiredTypeDefault valueDescription
serverYstring[]Location of the server (e.g. 'localhost' or an IP address)
portYnumber0Port to connect to the server on
numsensorsYnumber1Number of liberty receivers present
hemisphereN1 by 3 matrix[0,0,1]The hemisphere to use (should be a 1x3 vectors, e.g. [0 0 1] means use the +z hemisphere)
recordOrientationNnumber0Whether to record orientation as well as position (so each data sample has 6 numbers rather than 3)
samplerateNnumber240Sample rate (can be either 120 or 240 (in Hz))
alignmentframeN1 by 9 matrix[0,0,0,1,0,0,0,1,0]a 1x9 vector with the alignment frame. The first 3 are the zero position, the next three the direction of the x axis, the next three the direction of the y axis (in the coordinates of the transmitter cube
displayRangeXNn by 7 matrix[-10,10,0,0,0,0,0]Range of values to show when using showPosition as the dot position on the x axis. The ith row corresponds to the ith sensor. Use the first two values to show the liberty x values, next two for y, last two for z, e.g. [-10 10 0 0 0 0 0] to show x values of -10 to 10 as the dot x position. Use the last value to show time, e.g. [0 0 0 0 0 0 1] will move across the screen in 1 s
displayRangeYNn by 7 matrix[0,0,0,0,-10,10,0]Range of values to show when using showPosition as the dot position on the y axis. The ith row corresponds to the ith sensor. Use the first two values to show the liberty x values, next two for y, last two for z, e.g. [0 0 0 0 -10 10 0] to show z values of -10 to 10 as the dot y position. Use the last value to show time, e.g. [0 0 0 0 0 0 1] will move up the screen in 1 s.
offsetXNnumber0x offset when using showPosition
offsetYNnumber0y offset when using showPosition
showPositionColorN1 by 3 matrix[192,192,192]color to show dot when using showPosition (each number between 0 and 255)};
