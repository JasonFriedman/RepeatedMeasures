dots

A dot to be drawn in each frame

NameRequiredTypeDefault valueDescription
dotsFilenameYesstring[]A file containing the locations of the dots to draw (in the stimuli directory). It should be an Nx2 array (each line is the x and y coordinate)
flipNo1 by 2 matrix[0,0]whether to flip the x (1) or y (1) values
typeNonumber0type of dot, 0 = square, 1 = circle, 2 = high quality anti-aliased circle, 3 = ellipse
framesPerDotNonumber4number of frames to show each dot for
colorNo1 by 3 matrix[255,255,255]The color of the rectangle (RGB, 1x3 matrix, from 0 to 255)
sizeNomatrix10width of the dot (in pixels) (for types 1-3), or a 1x2 matrix (width and height) for type 4 (ellipse)
