pointToPoint

This response is to for (possibly repeated) point to point measurements.

NameRequiredTypeDefault valueDescription
startYnumber[0,0]The start point for the centre out movement (from those defined in setup.positions)
endYnumber[0,1]The end point for the centre out movement (from those defined in setup.positions)
startDistanceNnumber0.02How close to the start point you need to be to start a trial (inf for any distance), relative to screen width (1 = screen width)
endDistanceNnumber0.02How close to the end point you need to be to end a trial (inf for any distance), in pixels
repetitionsYnumber10Number of repetitions to perform. The trial will end when this number of repetitions has been reached (or the recordingTime has been reached)
beepNboolean0Whether to beep afer each successful movement
