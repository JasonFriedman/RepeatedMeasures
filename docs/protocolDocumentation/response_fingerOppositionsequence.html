fingerOppositionSequence

The response is touching the thumb and another finger in a sequence N times

NameRequiredTypeDefault valueDescription
sequenceYesmatrix[]The sequence to be performed (e.g. [4 1 2 3 4])
repetitionsYesnumber1The number of repetitions to be performed to complete the trial
fingermarkersYesmatrix[6,7,8,9,10]The markers which are the fingertips, first the thumb, then index, etc (e.g. [6 7 8 9 10])
touchThresholdNonumber2threshold below which fingers are considered touching
untouchThresholdNonumber3threshold above which fingers are no longer touching
fingertipdistanceNonumber2distance from the marker to the respective fingertip
